# Port

The Port feature creates a named harness connection point for wire-harness routing.

Ports are used by assembly components and the Wire Harness workbench to identify valid wire endpoints. A port stores its name, kind, and connection geometry so routed harness paths can resolve From and To endpoints.

## Typical Use
1. Add or import components that need harness connections.
2. Add ports at the intended wire endpoints.
3. Use spline paths and the Wire Harness panel to route wires between ports.

## Workbench Availability

Available in the Wire Harness workbench and the All workbench.
