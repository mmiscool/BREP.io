# `Face.surfaceArea()`

Sums the face's triangle areas in world space.

## Usage

```js
const area = face.surfaceArea();
```

## Signature

```js
face.surfaceArea()
```

## Parameters

None.

## Returns

`number` - Sum of triangle areas in world-space units squared.

This is the face-level measurement companion to `Solid.surfaceArea()`.
