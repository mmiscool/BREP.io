# `Solid._ensureFaceIndex()`

Internal helper that builds the face-ID-to-triangle lookup cache used by face queries.

## Usage

```js
solid._ensureFaceIndex();
```

## Signature

```js
solid._ensureFaceIndex()
```

## Parameters

None.

## Returns

`void`.

Most callers should use `getFace()` or `getFaces()` instead.
