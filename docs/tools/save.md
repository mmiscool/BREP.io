# Save

Saves the current model to the active storage location.

The saved model uses the file manager storage backend, which can be local browser storage or configured GitHub repo storage. 3MF saves include model geometry and embedded feature history metadata.

## Workbench Availability

Available in Modeling, Import, Surfacing, Sheet Metal, Assemblies, Wire Harness, PMI, Simulation, and All.
