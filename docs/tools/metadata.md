# `🏷️` Metadata

The Metadata window edits name-keyed metadata for the selected scene object.

![Metadata floating window](../floating-windows/metadata.png)

- The `🏷️` Metadata selection-context action appears when something is selected and toggles the panel.
- The panel follows the active selection, so selecting another face, edge, or solid refreshes the displayed metadata target.
- Entries can be added, edited, duplicated, deleted, searched, and bulk deleted from the table.
- Metadata color changes are reapplied to the viewport after edits when metadata coloring is active.

## Workbench Availability

Available as a selection context action when the current selection is not empty.
