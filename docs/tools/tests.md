# Browser Tests

Opens the Browser Testing window.

This is a development tool for running browser-side tests and inspecting test status without leaving the CAD page.

![Browser Testing floating window](../floating-windows/browser-testing.png)

## Workbench Availability

Localhost-only. Available in Modeling, Import, Surfacing, Sheet Metal, Assemblies, Wire Harness, PMI, Simulation, and All when development tools are enabled.
