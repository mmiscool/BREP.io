# `📥` Import

Opens the browser file picker for supported model inputs.

The import flow supports 3MF and BREP JSON. 3MF files with embedded history restore editable feature history when available; geometry-only 3MF imports load as model geometry.

## Workbench Availability

Available in Modeling, Import, Surfacing, Sheet Metal, Assemblies, Wire Harness, PMI, Simulation, and All.
