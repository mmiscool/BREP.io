# History Test Snippet

Generates a test snippet from the current feature history.

The generated snippet is useful for bug reports and regression tests because it captures the feature sequence needed to reproduce the current model state.

![Generated Test Snippet floating window](../floating-windows/history-test-snippet.png)

## Workbench Availability

Available in Modeling, Import, Surfacing, Sheet Metal, Assemblies, Wire Harness, PMI, Simulation, and All.
