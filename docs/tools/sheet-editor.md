# `🧾` 2D Sheet Editor

Opens 2D Sheets Mode for creating drawing-style sheets from the current model.

Use this tool to edit text, shapes, tables, images, and PMI view placements in the sheet editor.

## Workbench Availability

Available in Modeling, Import, Surfacing, Sheet Metal, Assemblies, Wire Harness, PMI, Simulation, and All.
