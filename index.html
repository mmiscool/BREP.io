BREP.io: The Open Source CAD for the Web

A web-based 3D modeling tool for engineers and makers. Key features include:

  • Full parametric constraint solver
  • B-Rep (Boundary Representation) geometry kernel
  • No installation required - runs in any modern browser
  • Export to STL and STEP for 3D printing and manufacturing
