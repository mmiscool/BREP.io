# brep-io-kernel examples (Node.js)

These examples use the published package `brep-io-kernel` in Node.js.

## Setup

```bash
npm i brep-io-kernel
```

## Run an example

```bash
node cube.js
node sphere.js
node cylinder.js
node cone.js
node pyramid.js
node torus.js
node boolean-union.js
```

Note: The package is ESM-only. These example files use ESM syntax, so run them
with Node.js 18+.
