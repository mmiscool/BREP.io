# brep-io-kernel examples (Node.js)

These examples use the published package `brep-io-kernel` in Node.js.

## Setup

```bash
npm i brep-io-kernel
```

## Run an example

```bash
tsx cube.ts
tsx sphere.ts
tsx cylinder.ts
tsx cone.ts
tsx pyramid.ts
tsx torus.ts
tsx boolean-union.ts
tsx license-info.ts
```

Note: The package is ESM-only. These example files are TypeScript, so run them
with `tsx` on Node.js 18+.
